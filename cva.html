Product Design

CVA: análise de mercado

Customer Value Analysis é uma metodologia de análise de mercado em que se atribui notas para critérios de avaliação e empresas. Este projeto foi o redesign desta funcionalidade tendo como base uma versão mais antiga e feedbacks específicos do cliente.

Desafio

O principal desafio se deu pelo fato de ser uma ferramenta bastante complexa. A compreensão do conceito, a definição do fluxo de navegação e criação de telas exigiram conversas e experimentações com protótipos de alta fidelidade até se atingir a entrega final. Além disso, o usuário da ferramenta são analistas especializados já acostumados com a metodologia o que aumentou bastante o grau de exigência.

Meu papel

Fui a única designer a trabalhar no projeto, portanto meu trabalho englobou a busca de requisitos com steakholders, a coleta de dados e ferramentas necessárias, consulta de documentos e versão anterior bem como a criação de entregáveis de design.
